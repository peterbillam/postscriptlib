NAME

      fonts.ps - Some useful fonts in PostScript

SYNOPSIS 

 (/home/wherever/ps/lib/fonts.ps) run
 % to send it to a printer see include_run . . .

 /TimesSmallCaps 30 selectfont
    50 700 moveto (TimesSmallCaps) show
 /TimesItalicSmallCaps 30 selectfont
    50 650 moveto (TimesItalicSmallCaps) show
 /TimesBoldSmallCaps 30 selectfont
    50 600 moveto (TimesBoldSmallCaps) show
 /TimesBoldItalicSmallCaps 30 selectfont
    50 550 moveto (TimesBoldItalicSmallCaps) show
 /HelveticaSmallCaps 30 selectfont
    50 470 moveto (HelveticaSmallCaps) show
 /HelveticaItalicSmallCaps 30 selectfont
    50 420 moveto (HelveticaItalicSmallCaps) show
 /HelveticaBoldSmallCaps 30 selectfont
    50 370 moveto (HelveticaBoldSmallCaps) show
 /HelveticaBoldItalicSmallCaps 30 selectfont
    50 320 moveto (HelveticaBoldItalicSmallCaps) show
 /Bitfont5x7 30 selectfont
    50 240 moveto (BITFONT5x7) show
 /FatSimple 30 selectfont
    50 160 moveto (FatSimple) show
 /FatSimpleItalic 30 selectfont
    50 110 moveto (FatSimpleItalic) show
 showpage

DESCRIPTION

This module implements, in PostScript, a small collection of custom fonts.

/TimesSmallCaps 30 selectfont

This is Times-Roman with lower-case letters replaced by smaller-size uppercase letters.
See Carlos in comp.lang.postscript on 20160522

/TimesItalicSmallCaps 30 selectfont

This is Times-Italic with lower-case letters replaced by smaller-size uppercase letters.

/TimesBoldSmallCaps 30 selectfont

This is Times-Bold with lower-case letters replaced by smaller-size uppercase letters.

/TimesBoldItalicSmallCaps 30 selectfont

This is Times-BoldItalic with lower-case letters replaced by smaller-size uppercase letters.

/HelveticaSmallCaps 30 selectfont

This is Helvetica with lower-case letters replaced by smaller-size uppercase letters.

/HelveticaItalicSmallCaps 30 selectfont

This is Helvetica-Oblique with lower-case letters replaced by smaller-size uppercase letters.

/HelveticaBoldSmallCaps 30 selectfont

This is Helvetica-Bold with lower-case letters replaced by smaller-size uppercase letters.

/HelveticaBoldItalicSmallCaps 30 selectfont

This is Helvetica-BoldOblique with lower-case letters replaced by smaller-size uppercase letters.

/Bitfont5x7 30 selectfont

This nostalgic 5x7 font has a strong cultural reference, and should become an important fashion-accessory. For example in green against a black background. It evolves from bitfont.ps by luser- -droog :
  github.com/luser-dr00g/xpost/blob/master/data/bitfont.ps
and has been modified mainly to use imagemask rather than image (so it respects the prevailing background and foreground colours), and has been given some more characters.
It so far has 0-9, A-Z, a-z and + - = . , : ; ! ? / \ ( ) [ ] { } & % @ ^ ~ $ _ * # > <

/FatSimple 30 selectfont

This is a Type 3 font, with simple shapes and very broad strokes. It creates very clear tangent-lines along the top and the bottom of the letters, which can be useful graphically. Its only letters are uppercase, including the ISO accents for german and french, plus the digits 0-9 and also + - = . : | ! ?
Remember charpath will not work with any Type 3 font.

/FatSimpleItalic 30 selectfont

This is the same as FatSimple but sloping strongly to the right.

INSTALL

To install: go to www.pjb.com.au/comp/free/fonts.ps.txt and save the file to your local disc.
Rename it to fonts.ps and move it into some appropriate directory such as ~/ps/lib . . .

Or, first change directory to where you keep your PostScript libraries:
    cd /home/wherever/ps/lib/
(or wherever)   and then either:
    wget -O fonts.ps http://www.pjb.com.au/comp/free/fonts.ps.txt
or:
    curl http://www.pjb.com.au/comp/free/fonts.ps.txt -o fonts.ps

Or, get it from github:
    git clone https://github.com/peterbillam/postscriptlib
